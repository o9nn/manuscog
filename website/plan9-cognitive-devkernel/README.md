# Plan 9 Cognitive DevKernel — Website

Interactive documentation website for the Plan 9 Cognitive DevKernel, a self-aware cognitive development kernel for Plan 9 from Bell Labs.

## Architecture

The website is built with React 19, Tailwind CSS 4, and Framer Motion, featuring a dark cybernetic "Dark Namespace" design theme.

### Sections

| Section | Description |
|---------|-------------|
| **Hero** | Animated cybernetic background with composition expression |
| **Composition Architecture** | 4-layer pipeline: Infrastructure → Cognitive → Domain Transform → Packaging |
| **Cognitive Namespace** | Interactive `/cognitive/` file tree with expandable directories |
| **Temporal Hierarchy** | 9-level time crystal architecture (8ms → 1000ms) |
| **Promise-Lambda Attention** | 8-promise kernel validation with animated status cards |
| **CPU Server Grid** | Distributed grid topologies, partitioned AtomSpace, CLI reference |
| **Development Environment** | QEMU devcontainer setup, port forwarding, IDE options |
| **Kernel Source Code** | Tabbed Plan 9 C source viewer with syntax highlighting |
| **Autognosis Self-Image** | Hierarchical self-monitoring with convergence simulation |

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```
